Controls

    Left Arrow  - Move the tile selector one tile left
    Right Arrow - Move the tile selector one tile right
    Up Arrow    - Move the tile selector one tile up
    Down Arrow  - Move the tile selector one tile down
    Left Mouse  - Select/place tile
